Text-Align

Text align is the horizontal alignment of text in an element. There is 4 different types of text alignment: Justifiy, left, right, and center.

  • Text-align: justify - This makes it so the spaces of the paragraph have equal width. It look like this:

    • Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce sit amet lectus vel justo aliquet dapibus id nec elit.

  • Text-align: center - This makes the text centered. It look like this:

    • Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce sit amet lectus vel justo aliquet dapibus id nec elit.

  • Text-align: right - Makes the text to right. It look like this:

    • Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce sit amet lectus vel justo aliquet dapibus id nec elit.

  • Text-align: left - Makes the text to left (the original). It look like this:

    • Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce sit amet lectus vel justo aliquet dapibus id nec elit.

    An "hr" element adds a horizontal line across the width of its containing element.
